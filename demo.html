Mouse Rover

Automatic Mouse Movement Tool

What is Mouse Rover?

Mouse Rover is a Python utility that automatically moves your mouse cursor at specified intervals. It's designed to keep your computer active and prevent it from going to sleep or showing as idle.

Installation

First, install the required dependencies:

$ pip install -r requirements.txt

Usage

View all available options:

$ python mouse-rover.py -h usage: mouse-rover.py [-h] [-a] [-s] [-t TIME] control for mouse-rover. optional arguments: -h, --help show this help message and exit -a, --always execute always. -s, --small move in small range. -t TIME, --time TIME trigger interval.

Command Options

  • -a, --always: Keep running continuously even if the user moves the mouse
  • -s, --small: Move the cursor in a small range around the current position
  • -t TIME, --time TIME: Set the interval (in seconds) between movements (default: 10 seconds)

Examples

Example 1: Move randomly every 10 seconds continuously
$ python mouse-rover.py -t10 -a
Example 2: Move in small range every 5 seconds
$ python mouse-rover.py -t5 -s -a
Example 3: Move once every 30 seconds (stops when user moves mouse)
$ python mouse-rover.py -t30

How It Works

Mouse Rover monitors your mouse position at regular intervals. If the mouse hasn't moved (indicating you're away or inactive), it automatically moves the cursor to a new position. The script includes smart features like:

  • Detecting and avoiding hot corners on your screen
  • Maintaining a safe border distance from screen edges
  • Logging all movements with timestamps
  • Option to continue running even when you're actively using the mouse
